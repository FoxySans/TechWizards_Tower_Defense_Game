Téma : Tower Defense Játék

Specifikáció : A játék során a játékos egy FPS nézetből fog egy karaktert irányítani és tornyokat építeni, hogy megvédje bázisát a betolakodóktól. A tornyokat kijelölt helyekre lehet majd építeni, és a toronyfajták variálásával lehet majd taktikázni. Az FPS nézet miatt a karakter rendelkezni fog saját fegyverrel, így a játékos aktívan részt tud venni a bázis védelmében.
